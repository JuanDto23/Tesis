\subsection{Tiempo de ejecución (\textit{Wall-clock time})}
\label{section:tiemejec}

Para el tiempo de ejecución, se consideran dos tipos. El \emph{tiempo de ejecución secuencial} $T_s$ se define como el tiempo transcurrido desde el inicio hasta el final de la ejecución de un programa secuencial. En el caso del \emph{tiempo de ejecución paralelo} $T_p$, este se define como el tiempo que transcurre desde el momento en que comienza el primer proceso hasta el momento en que finaliza la ejecución del último proceso. \cite{grama2003introduction}
