\subsection{Eficiencia}
\label{section:efic}

La \emph{eficiencia} es una medida de la fracción del procesamiento que se utiliza de forma útil, el cual se define como:

\[E = \frac{S}{p} = \frac{\left(\frac{T_s}{T_p}\right)}{p} = \frac{T_s}{p \cdot T_p}\]

Un programa paralelo con aceleración lineal ideal, tiene una eficiencia $E = 1$. \cite{pacheco2011introduction}
